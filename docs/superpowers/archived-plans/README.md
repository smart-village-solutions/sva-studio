# Archived Plans

Dieser Ordner enthält abgeschlossene Implementierungspläne aus `docs/superpowers/plans/`.

Ein Plan gehört hierher, wenn:
- alle inhaltlichen Umsetzungsschritte abgeschlossen sind,
- die Checkboxen im Plan auf dem tatsächlichen Stand gepflegt wurden,
- kein echter Rest-Backlog mehr im aktiven Plan verbleibt.

Der aktive Arbeitsordner bleibt `docs/superpowers/plans/`.
Abgeschlossene Pläne werden nach `docs/superpowers/archived-plans/` verschoben, damit der offene Backlog im aktiven Ordner sichtbar bleibt.
