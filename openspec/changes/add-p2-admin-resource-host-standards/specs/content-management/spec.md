## ADDED Requirements

### Requirement: Content-nahe Admin-Ressourcen verwenden gemeinsame Host-Standards

Das System SHALL für content-nahe Admin-Ressourcen dieselben hostgeführten Standards für Listen- und Revisionsverhalten verwenden.

#### Scenario: Content-Ressource zeigt hostgeführte Historie

- **WHEN** eine Content-Ressource Historie oder Revisionen anbietet
- **THEN** erfolgt deren Darstellung nach einem gemeinsamen Host-Muster
- **AND** das Fachpackage ersetzt dieses Muster nicht durch einen separaten Sonderflow
