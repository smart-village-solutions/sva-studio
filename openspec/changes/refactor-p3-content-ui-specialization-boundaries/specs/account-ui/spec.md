## ADDED Requirements

### Requirement: Spezialisierte Content-Views bleiben Teil des gemeinsamen Admin-Erlebnisses

Das System SHALL spezialisierte Content-Views innerhalb derselben Admin- und Account-UI-Grundmuster halten.

#### Scenario: Fachview respektiert Host-Standards

- **WHEN** eine spezialisierte Content-Ansicht im Studio angezeigt wird
- **THEN** bleibt sie visuell und interaktional Teil derselben Admin-Oberfläche
- **AND** sie verletzt nicht die hostgeführten Navigations-, Formular- und Statusmuster
